The Break-Even Concept

Break-even analysis is one of the most practical and frequently tested topics in GET 311. At its core, it answers a deceptively simple question: how many units must we produce and sell before we stop losing money? Below that output level every unit sold deepens the loss; above it every additional unit generates pure profit.

The analysis rests on a clean separation of costs into two categories: Fixed Costs, which are constant regardless of output (rent, depreciation, administrative overhead), and Variable Costs, which scale directly with each unit produced (materials, direct labour, packaging).

The Break-Even Point (BEP) is therefore defined as the output level Q at which Total Revenue = Total Cost: no profit is earned, no loss is suffered, and the business has exactly recovered all its costs for that period.

// CORE VARIABLES F = Total Fixed Costs (₦/period) p = Selling price per unit (₦/unit) v = Variable cost per unit (₦/unit) Q = Quantity produced (units) // COST AND REVENUE EQUATIONS Total Cost (TC) = F + v·Q Eq.26 Sales Revenue (SR) = p·Q Eq.27 // AT BREAK-EVEN: TC = SR F + v·Q_BEP = p·Q_BEP F = Q_BEP(p − v) Q_BEP = F / (p − v) Eq.28 ← Key // (p − v) is called the CONTRIBUTION MARGIN // Each unit sold contributes this to covering F

Extended Break-Even Concepts

Contribution, P/V Ratio & Margin of Safety

Contribution Margin

The amount each unit sold contributes towards covering fixed costs, after variable costs are paid. Once FC is fully covered, each unit contributes pure profit.

C = p − v // per unit At BEP: C × Q = F
Profit/Volume Ratio (ϕ)

The fraction of each naira of sales revenue that becomes contribution. A higher ϕ means faster recovery of fixed costs.

ϕ = (p − v) / p = Contribution / SR = F / (p·Q) Eq.34-36 Q_BEP = F / ϕ Eq.37
Margin of Safety (MoS)

How far current output is above the BEP. It measures business resilience — how much sales can fall before losses begin.

MoS = Q_current − Q_BEP MoS% = MoS / Q_current × 100 // Higher MoS% = safer business
Worked Examples — Including Past Paper Solutions
Example 1 — Prof. Bala's Classic (Sec. 5.5.5)

A company: output 5,000 items/month, fixed cost ₦2,000/month, variable cost 20k/item, selling price 40k/item. Find BEP and loss at 5,000 units.

// Note: 20 kobo = ₦0.20, 40 kobo = ₦0.40 p = ₦0.40/unit, v = ₦0.20/unit, F = ₦2,000 Q_BEP = F/(p−v) = 2000/(0.40−0.20) = 2000/0.20 = 10,000 units Sales at BEP = 10,000 × 0.40 = ₦4,000 // Loss at 5,000 units (below BEP): TC = 2000 + 0.20×5000 = ₦3,000 SR = 0.40 × 5000 = ₦2,000 Loss = TC − SR = ₦1,000

At 5,000 units (half the BEP), the company loses ₦1,000 — exactly the fixed cost minus contribution earned.

Example 2 — Past Paper Q63 (Exam Staple)

Manufacturer: labour ₦115/unit, material ₦76/unit, variable cost ₦2.32/unit, selling price ₦600/unit, monthly overhead ₦428,000.

// Step 1: Total variable cost per unit v = 115 + 76 + 2.32 = ₦193.32 // Step 2: Contribution per unit C = p − v = 600 − 193.32 = ₦406.68 // Step 3: BEP Q_BEP = F / C = 428,000 / 406.68 = 1,052.6 ≈ 1,053 units ← Answer D // Always round UP for BEP // (1,052 units still makes a loss)

Key trap: labour + material + variable overhead ALL count as variable costs.

Example 3 — Two-Process Comparison (Exercise Q2 from Notes)

Process X: F=₦20,000/yr, v=₦12/unit. Process Y: F=₦8,000/yr, v=₦22/unit. At what production quantity Q are total costs equal?

// Set TC_X = TC_Y 20,000 + 12Q = 8,000 + 22Q 20,000 − 8,000 = 22Q − 12Q 12,000 = 10Q Q = 1,200 units/year // Verify at Q=1,200: TC_X = 20,000 + 12×1,200 = ₦34,400 TC_Y = 8,000 + 22×1,200 = ₦34,400 ✓
Decision Rule
  • Q<1200
    Use Process Y — lower fixed cost dominates at low volumes
  • Q=1200
    Indifferent — both processes cost the same
  • Q>1200
    Use Process X — lower variable cost pays off at high volumes

Applications of Break-Even Analysis

  • ①
    Product Planning: Determine which new products to introduce and which existing products to discontinue based on their contribution margins and BEP requirements.
  • ②
    Product Pricing: Analyse how price changes affect BEP. A higher price reduces BEP units needed; a lower price may increase market share but requires higher volume to break even.
  • ③
    Profit Planning: Extend the BEP formula to find Q required for a target profit: Q = (F + Profit) / (p − v).
  • ④
    Make-or-Buy Decisions: Compare the total cost of internal manufacture versus external purchase. The BEP tells you the quantity at which manufacturing in-house becomes cheaper.
  • ⑤
    Equipment Selection: Compare two machines with different fixed and variable cost structures to find the production volume at which each is preferred — exactly the Process X vs Y analysis above.
  • ⑥
    Budget Planning: Express break-even as a percentage of budgeted sales to know when in the fiscal year/month/week the business goes from loss to profit.
⚑ Common Pitfalls — Break-Even Analysis
  • Missing a variable cost component: In Q63, some students forget the ₦2.32 variable overhead, giving a wrong v and wrong BEP. Add ALL variable costs: labour + material + variable overhead + royalties.
  • Treating royalties as fixed costs: Royalties paid per unit sold are variable costs (see Q66 — Lebron James royalty of ₦1,000 per pair is variable, making total v = 900+800+400+1000 = ₦3,100).
  • Not rounding UP: BEP = 1,052.6 means you need 1,053 units (selling 1,052 still results in a tiny loss). Always round up for BEP calculations.
  • Using revenue instead of contribution: The P/V ratio is contribution/revenue, not profit/revenue. Contribution = SR − Variable Costs, not SR − Total Costs.
  • Ignoring the linear model assumption: The standard BEP assumes constant prices and variable costs. In reality, bulk discounts, overtime labour, and price elasticity make the curves non-linear — but the exam assumes linearity.
Self-Assessment · Break-Even Analysis
Q1. A new product has monthly fixed cost ₦6,000, variable cost ₦2/unit, selling price ₦7/unit. How many units must be sold to break even?
A. 600 units
B. 857 units
C. 1,000 units
D. 1,200 units
✓ Answer: D — 1,200 units.
Q_BEP = F/(p−v) = 6,000/(7−2) = 6,000/5 = 1,200 units. This is Exercise Q1(a) directly from Prof. Bala's notes. Option B (857) is the mistake made when dividing 6,000 by 7 instead of by the contribution margin of 5.
Q2. A shoe manufacturer: labour ₦900, materials ₦800, variable costs ₦400, royalty ₦1,000 per pair, fixed charges ₦5,000,000/month, selling price ₦5,000/pair. BEP pairs/month?
A. 2,590
B. 2,632
C. 2,712
D. 2,890
✓ Answer: B — 2,632 pairs.
Total v = 900 + 800 + 400 + 1,000 = ₦3,100 per pair (royalty is variable!)
Contribution = 5,000 − 3,100 = ₦1,900/pair
Q_BEP = 5,000,000 / 1,900 = 2,631.6 → 2,632 pairs. This is Q66 from the past paper.
Q3. Using the Q1 problem, what is the profit or loss if 1,000 units are sold in a month?
A. Profit of ₦4,000
B. Loss of ₦1,000
C. Break-even — no profit or loss
D. Profit of ₦1,000
✓ Answer: B — Loss of ₦1,000.
SR = 7 × 1,000 = ₦7,000 | TC = 6,000 + 2×1,000 = ₦8,000 | Loss = TC − SR = ₦1,000.
Alternatively: we are 200 units below BEP (1,200 − 1,000 = 200). Loss = 200 × contribution margin = 200 × ₦5 = ₦1,000. Same result — the shortfall approach is often faster.
