The Present Worth Method

The Present Worth (PW) method converts every cash flow in a project — initial investment, annual operating costs, revenues, and salvage value — to their equivalent value at time zero (now). By expressing everything in today's naira, you create a level playing field for comparison.

If the PW of all cash flows is positive, the project earns more than the required minimum rate of return (MARR) and should be accepted. If negative, the project fails to cover the minimum acceptable return. If zero, the project earns exactly MARR.

The method is most powerful when comparing mutually exclusive alternatives — choose the option with the highest (least negative or most positive) present worth.

// Single sum discounting P = F(P/F, i%, n) = F/(1+i)ⁿ // Series present worth P = A(P/A, i%, n) // Mixed cash flows: sum all present worths PW = −Initial Cost + PW(revenues) − PW(costs) + PW(salvage) // Decision rule If PW > 0 → Accept (earns more than MARR) If PW < 0 → Reject If PW = 0 → Earns exactly MARR
Example — Past Paper Q15: Proposal Comparison

Proposal A: Build now for ₦400,000. Proposal B: Build smaller for ₦300,000 now + ₦200,000 extension at end of year 5. Interest = 20%. Which costs less?

PW of Proposal A = ₦400,000 PW of Proposal B: = 300,000 + 200,000(P/F, 20%, 5) = 300,000 + 200,000 × 0.4019 = 300,000 + 80,376 = ₦380,376 Savings = 400,000 − 380,376 = ₦19,624 ∴ Proposal B is cheaper ← Answer D
Example — Simple Discounting

How much to invest NOW at 6%/year to collect ₦12,000 in 7 years?

P = 12,000(P/F, 6%, 7) = 12,000 × 0.6651 = ₦7,981.20 // Meaning: ₦7,981.20 today @ 6% // equals ₦12,000 in 7 years
Capitalized Cost — Infinite Life Projects

For projects lasting forever (dams, roads, public infrastructure), the present worth of perpetual maintenance is:

P = A/i // Eq. from P/A as n→∞ CC = First Cost + A/i // Dam example: FC=₦2M, annual maint=₦50,000, i=8% CC = 2,000,000 + 50,000/0.08 = 2,000,000 + 625,000 = ₦2,625,000

Capitalized cost is also known as Life Cycle Cost — the answer to Q108.

⚑ Common Pitfalls — Present & Future Worth
  • Unequal lives comparison: Never compare PW of two alternatives with different lifespans directly. Use the Least Common Multiple (LCM) of the lives, or use Annual Worth method instead.
  • Forgetting salvage value: It's a positive inflow at year n. Students often omit it entirely, undervaluing the investment.
  • Wrong reference point: PW brings everything to t=0. FW brings everything to t=n. Both should give the same accept/reject decision — if they don't, you've made an error.
Self-Assessment · Present & Future Worth
Q1. What present sum will yield ₦1,000 at 10% per year for 5 years?
A. ₦683.01
B. ₦620.92
C. ₦651.34
D. ₦595.00
✓ Answer: B — ₦620.92. P = 1000/(1.1)⁵ = 1000/1.6105 = ₦620.92. This is stated directly in Prof. Bala's lecture notes as a worked example in Section 3.4(ii).
Q2. The present worth of cost for an infinite (perpetual) time period is called:
A. Annual Cost
B. Incremental Cost
C. Capitalized Cost
D. Operating Cost
✓ Answer: C — Capitalized Cost. CC = First Cost + A/i for perpetual maintenance. Also known as Life Cycle Cost. Q107 and Q108 in the past paper test this definition.
Q3. Present worth of two ₦100 payments at end of years 3 and 4 at 8% annual interest:
A. ₦150.56
B. ₦152.88
C. ₦153.89
D. ₦151.09
✓ Answer: B — ₦152.88. P = 100(P/F,8%,3) + 100(P/F,8%,4) = 100×0.7938 + 100×0.7350 = 79.38 + 73.50 = ₦152.88. This is Q48 from the past paper.
