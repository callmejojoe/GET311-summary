Rate of Return & MARR

The Rate of Return (ROR), also called the Internal Rate of Return (IRR), is the interest rate at which the present worth of all cash flows in a project equals exactly zero. It is the actual interest rate that the investment earns — the "yield" of the project.

The MARR (Minimum Acceptable Rate of Return) is the organisation's threshold. If ROR ≥ MARR, the investment earns at least the minimum required and should be accepted. If ROR < MARR, reject.

MARR Decision Rule
If ROR ≥ MARR → ✓ ACCEPT If ROR < MARR → ✗ REJECT If ROR = MARR → Indifferent (marginal) // PW method equivalent: If PW(MARR) > 0 → Accept If PW(MARR) < 0 → Reject

After-Tax Rate of Return (Q19, Q33)

Investor wants 7% after 42% tax. What pre-tax rate is needed?

After-tax = Pre-tax × (1 − tax rate) 7% = Pre-tax × (1 − 0.42) = Pre-tax × 0.58 Pre-tax = 7/0.58 = 12.07% ← Answer A

Miss Evilla gets ₦1,340 from bank, promises ₦1,500 at end of 9 months. Find discount rate.

Discount = 1500 − 1340 = ₦160 // Discount rate = d/F (on FUTURE value) d = (160/1500) per 9 months = 0.10667/0.75 (annualise) = 13.73% ← Answer D // KEY: discount rate uses FUTURE value // Interest rate uses PRESENT value // They give different %s for same deal!
Q41 — Bimonthly Compounding

₦1,000 becomes ₦1,608.44 in 4 years compounded bimonthly. Find nominal rate.

Bimonthly = 6 times/yr, 4×6=24 periods 1608.44 = 1000(1 + r/6)²⁴ (1 + r/6)²⁴ = 1.60844 (1 + r/6) = 1.60844^(1/24) = 1.02 r/6 = 0.02 r = 0.12 = 12.00% ← Answer B
⚑ Common Pitfalls — Rate of Return
  • Discount rate vs interest rate: Discount is computed on the FUTURE (face) value. Interest is computed on the PRESENT (principal) value. Same transaction gives different percentages.
  • Not adjusting for compounding period: Monthly rate = nominal/12. Quarterly rate = nominal/4. Always match rate period to payment period.
  • Accepting any positive ROR: ROR must exceed MARR, not just be positive. A 5% ROR with a 15% MARR is still a bad investment.
